Project Overview

AIris is an intelligent dating platform built with microservices and powered by AI. It aims to improve matchmaking accuracy, ensure authenticity, and offer a seamless user experience through advanced recommendation algorithms and real-time communication features.

Achievements

  • Designed a microservices-based architecture using Spring Cloud.
  • Established polyglot persistence with MySQL, MongoDB, Redis.
  • Developed personalized recommendation system using CF, MF, Transformers.
  • Built AI-based image moderation for user verification.

Challenges

  • Ensuring smooth inter-service communication with scalability.
  • Fine-tuning recommendation model accuracy in dynamic environments.
  • Optimizing real-time performance for chat and session management.
